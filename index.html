04. Projects

Award-winning

VisionMate: AI Accessibility Platform

VisionMate is a Thai-localized Flutter accessibility platform for visually impaired users. It combines on-device computer vision, depth estimation, OCR, voice interaction, AI chat, peer assistance, and an ESP32 companion device so users can understand surroundings, read text, detect obstacles, and request human help from one mobile app. The project won 1st Place at Thammasat School of Engineering Senior Project presentation. Watch demo video.

What It Does & Why It Matters

  • Real-time safety camera - detects nearby obstacles and danger zones with YOLO11s object detection and MiDaS depth estimation, then announces warnings through Thai TTS.
  • OCR and document reading - uses Google ML Kit text recognition and AI-assisted document analysis to help users read signs, labels, images, and files aloud.
  • Voice-first interaction - combines speech-to-text, text-to-speech, haptics, volume-button shortcuts, and accessible Thai UI so core workflows can be used without relying on sight.
  • AI assistant - routes Gemini requests through Firebase Cloud Functions to answer questions, analyze images, and keep paid API keys out of the client app.
  • Peer assistance - supports community features, chat, push notifications, and Zego-based video calls so users can request help from volunteers.
  • ESP32 companion device - integrates with an external camera board through UDP discovery, telemetry polling, wake-word events, and HTTP frame capture.

Engineering Contributions

  • Built the mobile platform with Flutter/Dart, Firebase Auth, Firestore, FCM, Storage, and role-based app flow for visually impaired users and volunteers.
  • Designed the ML service layer around isolated services for object detection, depth estimation, OCR, AI requests, TTS, local storage, and IoT communication.
  • Implemented a round-robin fusion pipeline that alternates YOLO11s and MiDaS inference every 500ms to reduce CPU spikes on mid-range Android devices.
  • Added local-first reliability with Isar local storage, SharedPreferences for small flags, secure storage/encryption for sensitive data, and lifecycle-aware camera handling.
  • Improved accessibility through Thai voice announcements, semantic labels, haptic feedback, large interaction targets, and flows designed around blind-user navigation.
  • Validated core logic with focused tests for ML math, navigation helpers, IoT telemetry parsing, data models, theme behavior, and app smoke coverage.
