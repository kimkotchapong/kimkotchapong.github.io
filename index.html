04. Projects

🏆 Award-winning

VisionMate: AI-Powered Assistive App for the Visually Impaired

AI-powered cross-platform mobile application designed to assist visually impaired users in daily navigation and communication. Built as a senior project at Thammasat University, winning 1st Place in the Electrical Engineering category.

Core Features

  • Real-time Depth Estimation — MiDaS model running on-device via TFLite for obstacle distance awareness and safe navigation alerts
  • Object Detection Pipeline — COCO-based detection identifying 80+ object categories in real-time for environmental understanding
  • Fusion Safety Pipeline — Novel system combining depth estimation and object detection data for intelligent navigation warnings
  • Gemini AI Assistant — Multimodal conversational AI with camera input support for scene description and Q&A
  • Community Platform — Real-time voice/video calling between visually impaired users and volunteers via WebRTC
  • SOS Emergency System — Smart routing to nearest volunteer with location sharing for immediate assistance

Tech Stack

Flutter Dart Firebase TensorFlow Lite MiDaS COCO Gemini API ARCore WebRTC Cloud Functions Firestore Google Cloud
